cmiv-dira

A model based iterative image reconstruction algorithm in DECT.

cmiv-dira is a Matlab implementation of a model based iterative image reconstruction algorithm in dual-energy computed tomography. The algorithm is developed by students and researchers associated with the Center for Medical Image Science and Visualization (CMIV) of the Link√∂ping University.



Aim: The primary aim is to improve the accuracy of radiation treatment planning in brachytherapy and external beam radiotherapy by providing estimates of elemental compositions of tissues. The secondary aim is to improve the quality of reconstructed images by suppressing beam hardening and scatter artefacts.



Status: The code structure is still not fully stable. Current releases are useful mainly for the code developers.
